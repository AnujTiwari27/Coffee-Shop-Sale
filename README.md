# Coffee-Shop-Sale  
This project features a fully interactive Excel Sales Dashboard designed to track and analyze key business metrics in a clear and visually appealing way. The dashboard brings multiple datasets together to help users quickly understand performance trends and make informed decisions.  

# Key Highlights

Dynamic Charts & Visuals – Automated bar charts, line charts, and KPIs update instantly based on the dataset.  
Slicers & Filters – Quickly explore sales by city, product category, month, or customer type.  
Clean UI Design – Simple color scheme, sharp layout, and easy-to-read visuals.  
Data Cleaning & Transformation – Raw data processed using formulas and structured tables for accurate insights.  

KPIs Included  

Total Sales  
Total Orders  
Average Order Value  
Top-Selling Items  
City-wise Performance  

# Insights You Can Get From This Dashboard

Which product categories generate the highest revenue  
Monthly/weekly sales patterns  
Best-performing cities or regions  
Customer-based sales breakdown  
What drives revenue growth over time  

# Tools & Techniques Used

Microsoft Excel  
PivotTables  
PivotCharts  
Slicers  
Conditional Formatting  
Data Cleaning with Excel formulas
